# 更新日志

## [2026-01-29] - 目录重构

### 主要更改

1. **项目结构重构**：
   - 创建了新的目录结构，将项目转变为图片处理服务中心
   - 新增 `public/assets` 目录用于存放通用静态资源
   - 新增 `pages` 目录用于存放各个功能页面
   - 为拼图工具创建了独立的目录 `pages/imgassli`
   - 预留了图片编辑器目录 `pages/img`

2. **功能模块**：
   - 拼图工具 (imgassli)：已完成迁移和配置
   - 图片编辑器 (img)：预留目录结构
   - 未来功能：预留扩展空间

3. **文件更新**：
   - 创建了根目录首页 `index.html`，作为所有功能的入口
   - 创建了项目索引文件 `INDEX.md`，提供详细的项目结构索引
   - 更新了 `README.md`，说明项目的新结构和功能
   - 更新了拼图工具的配置文件和页面文件

4. **技术改进**：
   - 优化了目录结构，提高了代码的可维护性
   - 为每个功能模块提供了独立的配置和构建环境
   - 改善了用户体验，提供了统一的入口页面

### 新增文件

- `index.html` - 根目录首页
- `INDEX.md` - 项目索引
- `pages/imgassli/index.html` - 拼图工具页面入口
- `pages/imgassli/script.js` - 拼图工具脚本
- `pages/imgassli/styles.css` - 拼图工具样式
- `pages/imgassli/package.json` - 拼图工具依赖配置
- `pages/imgassli/webpack.config.js` - 拼图工具构建配置

### 迁移文件

- 将原根目录的拼图工具相关代码迁移到 `pages/imgassli` 目录
- 将原 `assets/lib` 目录的第三方库迁移到 `public/assets/lib` 目录

### 未来规划
