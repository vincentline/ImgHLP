# PNG压缩网页实现计划

## 📋 项目概述

基于 `tinypng-lib` 库创建一个高质量的PNG压缩网页，压缩效果媲美TinyPNG。

## 🎯 核心需求

1. **压缩库**：使用 `tinypng-lib`（基于 imagequant WASM，压缩率≈70%）
2. **交互方式**：拖拽上传多个图片
3. **压缩控制**：每个图片独立滑块调整压缩质量
4. **进度显示**：压缩过程实时进度
5. **下载功能**：单个下载 + 批量打包下载
6. **设计规范**：遵循 `UI-DESIGN-SYSTEM.md`

## 📁 文件结构

```
docs/
└── png-compress/
    ├── index.html          # 主页面
    ├── styles.css          # 样式文件
    ├── script.js           # 主逻辑
    └── dist/               # 构建输出（tinypng-lib wasm）
```

## 🔧 技术方案

### 依赖安装

```bash
npm install tinypng-lib tinypng-lib-wasm jszip --save
```

### 核心API使用

```javascript
import TinyPNG from 'tinypng-lib';

// 压缩图片
const result = await TinyPNG.compress(file, {
  minimumQuality: 35,  // 最小质量
  quality: 88          // 目标质量 (0-100)
});

// 返回结果
// {
//   success: boolean,
//   file: File,           // 压缩后的文件
//   originalSize: number,
//   compressedSize: number,
//   rate: number,         // 压缩率
//   blob: Blob,
//   rateString: string
// }
```

## ⚠️ 注意事项

1. **WASM 加载**: tinypng-lib-wasm 需要正确配置 webpack
2. **大文件处理**: 使用 WebWorker 避免阻塞主线程
3. **内存管理**: 及时释放 Blob URL
4. **错误处理**: 优雅处理压缩失败情况

* 支持格式: PNG, JPG, WEBP

* 单次处理: 无限制

* 用户体验: 流畅、直观、美观
